# 🏥 Hospital Management System

A backend-based Hospital Management System developed using **Node.js**, **Express.js**, and **MySQL**. This project demonstrates database connectivity, modular routing, and CRUD operations for managing hospital-related information.

## 📌 Features

- Doctor Management
- Patient Management
- Staff Management
- Pharmacy Management
- MySQL Database Connectivity
- Modular Project Structure
- Express.js Backend

## 🛠️ Technologies Used

- Node.js
- Express.js
- MySQL
- JavaScript
- Git & GitHub

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/RakshithaTR-xyq/Hospital_project.git
```

Go to the project folder:

```bash
cd Hospital_project
```

Install dependencies:

```bash
npm install
```

Start the project:

```bash
node dbhospital.js
```
